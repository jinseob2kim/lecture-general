## 1) `case01_smc_gwas_signal_binary.png`

- Source file: `../smc-jihyunbaek/reports/figures/gwas_signal_binary.png`
- Purpose (용도):  
  약물반응 이진 outcome(예: 5점 기준 반응 유무)에서 SNP 신호가 어디에 분포하는지 한눈에 보여주는 GWAS 요약 그림.
- Experiment / Context (실험·맥락):  
  `smc-jihyunbaek`의 BOMICS 약물반응 연구(정신과 코호트, lithium/valproate 반응성 분석).
- Analysis method (분석 방법):  
  SNP별 연관검정 결과를 `-log10(P)`로 시각화한 Manhattan 계열 요약(염색체 parity 색상 구분 포함).
- Generated by (생성 주체):  
  원본은 `smc-jihyunbaek` 프로젝트 분석/보고 파이프라인(차라투 분석팀 운영 코드)에서 생성된 보고용 figure.
- Concepts shown (그림 개념):  
  다중 SNP 후보 신호의 상대적 강도, trait별 신호 패턴 차이, 후보 hit의 우선순위 탐색.
